# 企业工商信息查询

## 功能概述

通过阿里云 API 市场提供的企业工商数据查询服务，获取企业的工商注册信息。

## 使用场景

- 企业背景调查
- 合作伙伴资质核实
- 投资决策支持
- 风险控制管理
- 客户信息验证

## 使用方法

### 方式一：直接调用脚本

```bash
python scripts/query_company.py "企业名称"
```

### 方式二：在代码中调用

```python
import subprocess
import json

result = subprocess.run(
    ['python', 'scripts/query_company.py', '北京百度网讯科技有限公司'],
    capture_output=True,
    text=True
)
data = json.loads(result.stdout)
```

## 输入参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| company_name | str | 企业名称、注册号、统一信用代码、组织机构代码任选其一 | 北京百度网讯科技有限公司 |

## 返回信息

查询成功时返回以下字段：

- 公司名称
- 公司历史名称
- 统一信用代码
- 组织机构代码
- 注册号
- 税号
- 公司类型
- 法定代表人
- 成立时间
- 核准时间
- 营业期限截止时间
- 注销时间
- 注册资本
- 注册地址
- 所属行业
- 行业分类

## 错误处理

- `50002`：未查询到企业信息（企业名称可能不准确）
- 其他错误码：查询失败

## 配置说明

脚本使用阿里云 API 市场的企业工商数据查询服务，需要配置：

1. 阿里云账号
2. API 市场的 AppCode（用于认证）

AppCode 配置位置：`scripts/query_company.py` 文件中的 `appcode` 变量

## 注意事项

1. 确保使用有效的 AppCode 进行认证
2. 企业名称输入需准确完整
3. 遵守 API 调用频率限制
4. API 返回数据基于官方工商登记信息
